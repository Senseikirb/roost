Module descriptions, AI prompt examples, and integration tips.

Module Descriptions
Risk Register
5x5 criticality matrix, rack/assembly assignment, formal action tracking, mitigation history, risk dashboard with heatmap.
Action Tracker
Track action items with owners, due dates, priority, status. Timeline and calendar views. Paste import from spreadsheets.
BOM Manager
Bill of Materials with part numbers, quantities, vendors, costs. BOM cost rollup across assemblies.
Inventory
Equipment tracking with serial numbers, locations, calibration dates, and status monitoring.
HW/SW Tracker
Hardware and software configuration items with version tracking, lifecycle status, and system node mapping.
EVM Tracker
Earned Value Management with BAC, PV, EV, AC. Calculates CPI, SPI, EAC, VAC. S-curve visualization.
Change Control
Engineering Change Proposals with classification, impact assessment, CCB status tracking.
Test Matrix
Verification and validation tracking with TADI methods, pass/fail results, coverage metrics.
Lessons Learned
Capture knowledge from successes and challenges. Categorized by phase and sentiment.
Requirements
Requirements traceability matrix with verification status, linked tests, and parent/child relationships.
Analysis and Trade Studies
Alternatives analysis with evaluation criteria, decision rationale, linked requirements and risks.
Anomaly Tracker
Track anomalies with severity levels, root cause analysis, corrective actions, and resolution status.
Cost Tracker
WBS-based budgets, actuals, committed, ETC, variance, category rollups.
Procurement
Unified Purchase and Delivery tracker. Flow items through Needed → Requested → Ordered → Shipped → Received with vendor rollups and overdue detection.
Links & Specs
Bookmarks and external references, plus a right-side Specifications sidebar to save document numbers (MIL-SPEC, SOPs, drawing #s) with descriptions.
AI Prompt Examples

Export your data as JSON, then use these prompts with AI assistants to analyze it:

"Here is my PMD export JSON. Analyze my risk register: identify the top 5 risks by criticality score, flag any overdue mitigations, and suggest risk response strategies for Critical items."
"Review my action tracker data. Generate a summary of overdue items grouped by owner, and draft follow-up messages for each owner listing their open items."
"Analyze my EVM data. Calculate program-level CPI, SPI, EAC, and TCPI. Flag any work packages with CPI or SPI below 0.9 and recommend corrective actions."
Integration Tips
  • Export JSON regularly as backup -- this tool has no server or cloud storage
  • Use Paste Import (on Action Tracker) to bulk-import rows from Excel/Sheets
  • Link risks to actions using the "Linked Risk" field for traceability
  • Use the Data Integrity Check on the dashboard to find orphaned references
  • The Cascade Analysis tool shows how risks flow to actions, BOMs, and anomalies
  • Print any module view using Ctrl+P -- print styles hide nav and controls
  • Email the .html file directly -- it works offline with no dependencies
